Домашнее задание
❑ написать свою реализацию связного списка, должны быть реализованы
методы add, remove, indexOf, addAt, removeAt;

Написать реализацию связанного списка.
Должен быть узел.
В нем должны
	храниться данные
	присутствовать ссылка на другой элемент списка.
Реализовать методы
	добавление элемента
	удаление элемента
	получение индекса по элементу
	добавление по определенному индексу
	удаление узла

Подсказка

1. Добавление элемента в начало списка:

Чтобы добавить новый элемент в начало списка, нужно присвоить свойству next нового элемента текущий первый элемент списка. Вот пример:

let list = {
value: 1,
next: {
value: 2,
next: {
value: 3,
next: {
value: 4,
next: null
}
}
}
};
